# Brijwasi - High-End Restaurant Landing Page

A premium, modern landing page for Brijwasi, featuring elegant design, smooth animations, and a seamless user experience.

## 🌟 Features

- **Premium Design**: Sleek, high-end aesthetics tailored for a luxury dining experience.
- **Dynamic Animations**: Powered by Framer Motion for sophisticated scroll-linked effects and transitions.
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices.
- **Modern Tech Stack**: Built with React, Vite, Tailwind CSS, and Shadcn UI.
- **Interactive Menu**: Sophisticated menu presentation with dietary indicators.
- **Gallery & Contact**: Dedicated sections to showcase the restaurant's ambiance and facilitate bookings.

## 🚀 Tech Stack

- **Frontend**: React.js
- **Styling**: Tailwind CSS, Vanilla CSS
- **Components**: Shadcn UI
- **Animations**: Framer Motion
- **Build Tool**: Vite

## 📁 Project Structure

- `frontend/`: Contains the main application code (React, styles, assets).
- `readme/`: Project documentation and details.

## 🛠️ Installation & Setup

1. **Navigate to the frontend directory**:
   ```bash
   cd frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
