## 🖥️ Website

This repository contains the source code for the **Brijwasi Sweets & Restaurant** official website — a modern, responsive, and visually stunning web application.

### Tech Stack

| Technology | Purpose |
|---|---|
| **React 19** | UI Framework |
| **Next.js / Vite** | Build & Dev Server |
| **TypeScript** | Type Safety |
| **Framer Motion** | Animations & Transitions |
| **Tailwind CSS** | Styling |
| **Lucide React** | Icons |

### Key Features

- 🎬 **Cinematic Hero Section** with video backgrounds
- 📱 **Fully Mobile Responsive** design optimized for all screen sizes
- 🍽️ **Interactive Menu** with 16 scrollable categories and 140+ items
- 🖼️ **High-Quality Food Photography** for each category
- 🔗 **Deep Linking** — navigate directly to specific menu categories
- ⚡ **Smooth Animations** powered by Framer Motion
- 📍 **Contact Integration** — direct call, email, and Zomato ordering links

### Pages

| Page | Description |
|---|---|
| **Home** | Hero video, heritage section, specialities grid, signature menu |
| **Menu** | Full 16-category interactive menu with category navigation |
| **Gallery** | Photo gallery showcasing dishes and ambiance |
| **About** | Restaurant history and story |
| **Contact** | Location, phone, email, and working hours |

## 🚀 Getting Started

```bash
# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:5173`
